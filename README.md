# spring-crud-app-rp3
RP3-クラウド環境構築演習でフォーク専用リポジトリです。このアプリをフォークして設定を書き換えてください。なお本ブランチにはプッシュできません。

## フォークとプロジェクトインポート
フォーク、インポートはEclipseを使って行ってください。Git手順書を参考に行います。

## 設定ファイル
application.propertiesに利用するプロファイルの記述があります。デプロイ時はawsに変更してください。
application-aws.propertiesのDB接続情報を変更してください

## デプロイファイル
{pj}/target/xxxx.war をAWS環境にアップロードし動作させてください。詳細は講師からの説明を聞いてください。
